<h1 align="center">Doffed Kanata</h1>

Replaces all of the following with dof-alternative:
- `defcfg`
- `defsrc`
- `deflocal*`
- `deflayer`
- `defalias`
- `defvar`
- `defseq`
- `deffakekeys`
- `defchords`

Learn about kanata [here](https://github.com/drdilyor/doffed-kanata)
